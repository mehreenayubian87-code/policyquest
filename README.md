# PolicyQuest

PolicyQuest is a Next.js MVP for running structured policy co-design workshops.

Version 1 includes:

- Health Policy, Education Policy, and Environment Policy categories
- Five policy issues
- A four-stage co-design workspace
- Stakeholder cards, activity cards, event cards, and resource tokens
- A final output page for a service improvement concept

## Run locally

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```
